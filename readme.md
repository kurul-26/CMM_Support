# CMM Support v1.0

🔧 CMM (Coordinate Measuring Machine) için açısal ölçüm hesaplama aracı.

## 📋 Özellikler

- ✅ Dikey (A) ve Yatay (B) model açısı hesaplama
- ✅ İç ve dış ölçüm desteği
- ✅ Görsel açı göstergesi
- ✅ Kayıt tutma sistemi


## 🚀 Kurulum

### Windows Kullanıcıları

1. github.com/kurul-26/CMM_Support/releases sayfasından en son sürümü indirin
2. `CMM_Support.exe` dosyasını çalıştırın
3. Kurulum gerektirmez!

### Geliştirici Kurulumu

```bash
# Depoyu klonlayın
git clone https://github.com/kurul-26/CMM_Support.git
cd CMM_Support

# Sanal ortam oluşturun
python -m venv venv
source venv/bin/activate  # Linux/Mac
# veya
venv\Scripts\activate  # Windows

# Bağımlılıkları yükleyin
pip install -r requirements.txt

# Uygulamayı çalıştırın
python src/CMM_Support_v1.0.py
```

## 📖 Kullanım

1. **Karakteristik & Item Numarası** girin
2. **Dikey Model Açısı (A)** değerini girin
3. **Yatay Model Açısı (B)** değerini girin
4. **İç/Dış Ölçüm** seçimini yapın
5. **Kaydet** butonuna tıklayın

## 🛠️ Teknolojiler

- Python 3.13
- PyQt6
- PyInstaller
